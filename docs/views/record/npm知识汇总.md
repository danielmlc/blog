#### 下载包
```bash
npm install xxxx -S  -D

```

#### 添加账号
```bash
npm adduser
npm whami   #查询当前账号
```

#### 发布
```bash
npm publish
```

#### 强制移除发布的npm包

```bash
npm unpublish xxxx --force
```
